# Music Playlist Extractor

Able to parse Amazon Music playlists and albums.

- Display song details in separate columns:
  - Song Name
  - Artist Name
  - Album Name
  - Duration

## How to Use

1. Enter a playlist or album URL into the text box. (supported URLs examples: `https://music.amazon.com/playlists/B08BWK8W15`, `https://music.amazon.com/albums/B001230JXC`)
2. Click the "Parse" button.
3. The playlist and song information will be displayed in the list below.
